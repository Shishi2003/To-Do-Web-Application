# SkillCraft Technologies – Web Development Internship

## Task 04: To-Do Web Application

This project is a multi-list to-do web application developed as part of the SkillCraft Technologies Web Development Internship.

---

## 🔧 Technologies Used
- HTML5  
- CSS3  
- JavaScript  

---

## ✨ Features
- Create multiple to-do lists
- Add tasks to individual lists
- Mark tasks as completed with strike-through effect
- Delete individual tasks
- Delete entire to-do lists
- Task completion counter for each list
- Glassmorphism and neon-inspired UI
- Smooth entry animations
- Data persistence using localStorage
- Responsive layout for desktop and mobile devices

## 📌 Description
The to-do web application allows users to create and manage multiple task lists in a clean and organized interface. Each list is displayed in a separate card with its own tasks and completion status. The project focuses on JavaScript DOM manipulation, event handling, and modern CSS styling techniques without using any external libraries or frameworks.
